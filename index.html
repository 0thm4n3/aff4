AFF4: The Advanced Forensic File Format.

AFF4 was introduced initially in the paper "Extending the advanced forensic format to accommodate multiple data sources, logical evidence, arbitrary information and forensic workflow" M.I. Cohen, Simson Garfinkel and Bradley Schatz, digital investigation 6 (2009) S57â€“S68.

What is it?

AFF4 is more than just a file format. AFF4 defines a complete Information Model for dealing with forensic evidence. AFF4 introduces a number of Primitive Objects that allow for the efficient storage and manipulation of digital evidence.

At its core the AFF4 is a library - available here in C++ and Python.
